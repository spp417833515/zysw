> 更新日期: 2026-04-03

## 二、详细步骤

### ① 系统生成财务报表（每季度末）

1. 进入系统「报表」→「税务报表」
2. 选择季度区间（如 2026-01-01 至 2026-03-31）
3. 点击「生成报表」，系统自动生成：
   - 资产负债表
   - 利润表
   - 现金流量表
4. 下载 Excel 文件备存

**关键数据来源：**

| 报表项目 | 取数逻辑 |
|:--|:--|
| 营业收入 | Transaction.type=income 全部（权责发生制） |
| 营业支出 | Transaction.type=expense 全部（权责发生制） |
| 利润总额 | 营业收入 - 营业支出 |
| 现金流入 | Transaction.type=income & payment_confirmed=True |
| 现金流出 | Transaction.type=expense & payment_confirmed=True |
| 账户余额 | accounts.balance |

### ② 核对数据

**必检项目：**

| 检查项 | 验证方法 |
|:--|:--|
| 利润总额 | = 收入合计 - 支出合计（不是账户余额！） |
| 现金流净额 | = 账户余额（如不等需排查未确认交易） |
| 应付职工薪酬 | 检查是否有未发工资、少付差额 |
| 发票匹配 | 已开发票金额 vs 已记录收入 |

**常见错误：**
- ❌ 利润总额用了账户余额（差额 = 未付款项）
- ❌ 企业所得税用 25% 标准税率（应用 5% 小微优惠）
- ❌ 现金流包含未实际收付的款项

### ③ 网上申报（电子税务局）

登录：**河南省电子税务局** → 我要办税 → 税费申报

#### 3.1 增值税申报

| 字段 | 填写规则 | 示例（Q1 2026） |
|:--|:--|:--|
| 销售额（1%征收率） | 不含税销售额合计 | 100,000.00 |
| 应纳税额 | 销售额 × 3%（系统自动按法定税率算） | 2,970.30 |
| 减免税额 | 3%→1% 减免 + 普票免税部分（系统自动） | 2,009.90 |
| 本期应缴 | 仅专票部分 × 1% | 990.10 |

**增值税规则：**
- 小规模纳税人征收率：1%（减按，原 3%）
- 季度销售额 < 30 万：普通发票免征增值税
- 专用发票：必须缴纳，不享受免征

#### 3.2 附加税申报

| 税种 | 计算方式 | 示例 |
|:--|:--|:--|
| 城市维护建设税 | 增值税 × 5%（县级）× 50%（小规模减半） | 24.75 |
| 教育费附加 | 季度销售额 < 30 万免征 | 0.00 |
| 地方教育附加 | 季度销售额 < 30 万免征 | 0.00 |

#### 3.3 企业所得税（月季报）

| 字段 | 填写规则 | 示例 |
|:--|:--|:--|
| 季初从业人数 | 季度初在职员工数 | 1 |
| 季末从业人数 | 季度末在职员工数 | 2 |
| 季初资产总额（万元） | 季初资产负债表资产合计 / 10000 | 0 |
| 季末资产总额（万元） | 季末资产负债表资产合计 / 10000 | 8.71 |
| **属于国家限制或禁止行业** | **必须选「否」** | 否 |
| 利润总额 | 收入 - 支出（不是账户余额） | 87,041.65 |
| 应纳所得税额 | 利润 × 25%（系统自动） | 21,760.41 |
| **减免所得税额** | 利润 × 20%（小微优惠，系统自动） | 17,408.33 |
| 实际应缴 | 利润 × 5% | 4,352.08 |

**⚠️ 小微企业认定条件（全部满足）：**

| 条件 | 标准 | 本公司 |
|:--|:--|:--|
| 年应纳税所得额 | ≤ 300 万 | ✅ |
| 从业人数 | ≤ 300 人 | ✅ |
| 资产总额 | ≤ 5000 万 | ✅ |
| 非限制/禁止行业 | 选「否」 | ✅ |

**附报事项：** 必须勾选「小型微利企业」，否则无法享受 5% 优惠税率

#### 3.4 利润总额明细弹窗

点击「利润总额本年累计金额」弹出明细表：

| 项目 | 金额 | 说明 |
|:--|--:|:--|
| 营业收入 | 101,005.00 | Transaction.type=income 合计 |
| 营业成本 | 0.00 | 无直接成本 |
| 管理费用 | 13,963.35 | 全部支出（含工资） |
| 研发费用 | 0.00 | 无则填 0 |
| 财务费用 | 0.00 | 无则填 0 |

### ④ 缴纳税费

**缴费方式（优先级）：**

1. **对公账户** → 银联缴款（推荐，最规范）
2. **个人代付** → 微信/支付宝扫码（方便快捷）
   - 保留完税证明
   - 系统记账时 payment_account_type 设为 `personal`

### ⑤ 公司报销（如个人代付）

1. 系统中创建支出交易：
   - 分类：税费
   - 金额：实际缴纳金额
   - payment_account_type: `personal`（不影响公司账户余额）
2. 待公司账户有资金时，创建报销批次
3. 报销完成后交易自动改为 `company` 并更新账户余额

- 时间：次年 5 月 31 日前
