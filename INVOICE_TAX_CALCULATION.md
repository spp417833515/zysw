# 税款计算说明 - 已开票与未开票

## 收入分类

系统现在会自动区分两种收入类型：

### 1. 已开票收入
- 标记为 `invoiceIssued` 或 `invoiceCompleted` 的收入
- 开票金额已包含增值税（含税价）
- 需要从含税金额中分离出增值税

**计算公式：**
```
增值税 = 已开票收入 / (1 + 税率) × 税率
```

**示例：**
- 已开票收入：103,000元（含税）
- 税率：3%
- 增值税 = 103,000 / 1.03 × 0.03 = 3,000元

### 2. 未开票收入
- 未标记开票状态的收入
- 按不含税金额计算增值税

**计算公式：**
```
增值税 = 未开票收入 / (1 + 税率) × 税率
```

## 税款计算逻辑

### 增值税
```
总增值税 = 已开票增值税 + 未开票增值税
```

### 附加税
```
附加税 = 总增值税 × 12%
```

### 企业所得税
```
企业所得税 = (季度利润 × 4) × 税率 / 4
```

## 免征判断

如果季度总收入 ≤ 30万元：
- 免征增值税
- 免征附加税
- 企业所得税正常计算

## 界面显示

仪表盘会显示：
1. **季度总收入** - 所有收入的总和
2. **已开票收入** - 已开具发票的收入（蓝色）
3. **未开票收入** - 未开具发票的收入（橙色）
4. **季度支出** - 所有支出的总和

税款明细会显示：
- 增值税：分别显示已开票和未开票的金额及对应税额
- 附加税：基于总增值税计算
- 企业所得税：基于季度利润计算

## 注意事项

1. 开票后增值税是确定要缴纳的
2. 未开票收入也需要申报增值税
3. 建议及时标记开票状态，以便准确计算税款
4. 季度收入接近30万时，注意控制开票金额
